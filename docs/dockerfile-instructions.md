# 📘 Most Used Dockerfile Instructions

---

## Categories of Dockerfile Instructions

### 1. Base Image
- `FROM`

### 2. File System Setup Inside Image
- `WORKDIR`
- `COPY`
- `ADD`

### 3. Build Time Execution
- `RUN`

### 4. Runtime Configuration
- `ENV`
- `ARG`

### 5. Container Startup Behavior
- `CMD`
- `ENTRYPOINT`

### 6. Networking
- `EXPOSE`

### 7. Metadata
- `LABEL`
